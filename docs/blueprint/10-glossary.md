# Part X — Glossary

**Canonical key**: The internally defined identifier that uniquely names an instrument independently of any one vendor or broker session.

**Carry**: A generic term covering financing and distribution effects that cause forward to differ from spot.

**Confidence score**: A quality label or numeric score indicating how trustworthy a derived estimate is.

**Derived analytics**: Any object computed from raw observations, such as forwards, IV points, surfaces, Greeks, or scenarios.

**Immutable raw layer**: The append-only storage layer containing normalized observations exactly as captured or reconstructed.

**Market-state snapshot**: A time-aligned, deterministic view of underlying and option quotes used by analytics modules.

**No-arbitrage diagnostic**: A check designed to flag surface shapes that are economically implausible or inconsistent.

**Replay**: The process of recomputing analytics historically from stored raw data or snapshots using the same code path as live processing.

**Scenario version**: A named, version-controlled set of stress definitions used to compute scenario PnL.

**Surface grid**: A regularized table of volatility or total-variance values sampled from the fitted surface representation.
